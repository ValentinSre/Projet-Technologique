Exemples

Inspire-toi de ces exemples

Ces exemples reviennent sur les requêtes basiques en s'inspirant des tables disponibles dans la base de données.

Projection

Sélection de toutes les colonnes de la table livre :

SELECT * FROM livre

Sélection des colonnes Titre et Annee de la table livre :

SELECT Titre, Annee FROM livre

Sélection des colonnes Titre de la table livre où le Genre est Comics :

SELECT Titre FROM livre WHERE Genre = "Comics"

Jointure

Joindre deux tables (forme générique):

SELECT * FROM table_A JOIN table_B ON tableA.clé = table_B.clé

Dans notre exemple, l'oeuvre de la table adaptation correspond au titre de la table livre.

SELECT * FROM livre JOIN adaptation ON livre.titre = adaptation.oeuvre

AIDE

Construis ta requête SQL

Utilise ce guide pour structurer ta requête de la meilleure manière possible !

  • Structure globale d'une requête

    SELECT colonne(s)
    FROM Table1
    JOIN Table2 ON condition(s)
    WHERE condition(s)
    GROUP BY colonne(s)
    HAVING condition(s)
    ORDER BY colonne(s) ASC/DESC

  • Création de table

    CREATE TABLE nomTable (nomColonne1 type, nomColonne2 type, ...)

  • Remplissage de table

    INSERT INTO Table VALUES (valeur1, valeur2, ...)

  • Fonctions d'agrégation utiles

    SUM() : calcule la somme d'une colonne ou groupe de lignes
    AVG() : calcule la moyenne d'une colonne ou groupe de lignes
    MAX() : retourne le résultat maximum
    MIN() : retourne le résultat minimum
    COUNT() : compte le nombre de lignes dans un résultat
    ROUND() : arrondit la valeur

  • ORDER BY

    Permet de trier les lignes (ASC = ascendant ; DESC = descendant)

  • GROUP BY

    Permet de grouper plusieurs résultats et utiliser une fonction sur un groupe de résultat

  • LIMIT

    Permet de spécifier le nombre maximum de résultats que l'on souhaite obtenir

  • HAVING / WHERE

    HAVING permet de filtrer après le GROUP BY alors que WHERE filtre avant
