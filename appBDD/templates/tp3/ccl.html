Conclusion



À travers ces quelques séances de travaux pratiques, nous avons donc vu ce que voulait dire le terme de donnée, notion essentielle dans l'utilisation du Web puisque les données composent le Web et affluent.

Nous avons découvert le Web des données qui présente une structure différente de la donnée puisque celles-ci sont structurées : il est ainsi possible de manipuler les données grâce aux liens entre ceux-ci.
