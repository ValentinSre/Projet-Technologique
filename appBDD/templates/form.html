Internet



Internet est un ensemble d’ordinateurs reliés entre eux. Il s’agit donc d’un réseau d'ordinateurs qui permet la communication entre les appareils aux quatre coins du monde.


Quel lien entre Internet et Web ?





Internet et le Web



Le Web est à ne pas confondre avec Internet. Les deux sont différents et distincts, Internet étant le support physique du Web.

Internet est un réseau informatique que le Web utilise tout comme d'autres services, comme les courriels (emails) ou le transfert de fichiers.
