Le SQL

Il existe différents types de bases de données. Dans le cadre de ce projet, nous nous intéresserons aux bases de données relationnelles, c'est-à-dire des bases de données qui organisent les données selon des tables, liées entre elles par des relations.








Le SQL

Explication

Le SQL (Structured Query Language) est un langage permettant de communiquer avec une base de données.
Ce langage est notamment très utilisé pour communiquer avec les données d’un site web.

Deux types de commandes sont essentielles en SQL : la projection (sélectionner un ensemble de colonnes dans une table) et la jointure (lier deux tables).
Nous allons pratiquer ces deux commandes dans les prochains exercices...
