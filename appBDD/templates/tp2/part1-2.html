URI

L'URI (Uniform Resource Identifier) permet d'identifier une entité utilisée sur le web. Les URI peuvent être reliées entre elles.

Il s'agit d'une extension de l'URL (qui permet d'identifier les pages du web uniquement). Ainsi, tout ce qui est identifié par une URI est une ressource du web accessible sur le web.
Cela rassemble donc les documents du web (identifiés par leur URL) mais aussi toutes les choses dont on parle sur le web, par exemple une table, une voiture, un humain ou encore une idée, un concept abstrait... Si l'on schématise, l'URI correspond à la concaténation entre l'URL pour localiser une ressource et de l'URN (Uniform Resource Name) qui est un identifiant unique pour une ressource.

Résumé

L'URL permet de localiser une ressource dans le web.
L'URN permet de référencer une ressource sur le web sans la localiser.
L'URI est l'ensemble de ces deux paramètres.
